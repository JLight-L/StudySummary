1. 创业精神
    1. 想法新
    2. 敢于面对风险
    3. 坚定，不被困难打倒
    4. 做事灵活，善于变通
2. 商业管理
    1. 商业模式是企业创造、传递和获取价值的系统逻辑，明确了客户、产品 / 服务与盈利方式。
    2. 它能指引企业方向、优化资源配置、构建竞争壁垒，保障盈利的可持续性。
    3. 主流类型包括产品销售、订阅制、平台型等，各有其核心运作特征。
    4. 商业模式必须**随市场形势动态调整**，固守不变会导致企业与市场脱节，最终被淘汰。
3. 精益创业
    1. 低成本极简产品 → 核心客户反馈 → 产品快速迭代适应市场
    2. 四个阶段：
        1. 商业模式的探索
        2. 阶段式探索
        3. 商业模式确立
        4. 商业模式的正常执行
4. 创业团队
    1. 高效创业团队原则：
        1. 目标明确、方向一致
        2. 人员特长互补，不冲突
        3. 要有权力中心
        4. 股权分配要合理
        5. 分工明确、各司其职
5. 创业机会来源
    1. 市场驱动：消费者痛点、消费者潜在需求；
    2. 技术驱动：人工智能、大数据等技术重构行业；
    3. 政策与人口变化：如老龄化催生健康产业机会；
    4. 竞争格局变化：行业竞争格局变化给新进入者带来机会。
6. 融资
    1. 融资主要方式：
        1. 债券融资（借钱）：银行贷款，发行债券，民间借款；风险：担保风险，财务风险
        2. 股权融资（出卖股权）：公开市场发售，私募发售；风险：有可能被资本方踢出局
7. 新公司立足
    1. 避免被行业巨头视为竞争者
    2. 找到自己的特点（差异化）
    3. 需要额外吸引力（提供新用户增长），如高性价比
        - *注*：就像王者出新英雄想要有人玩就得超标，或者就是特别有特色
8. 企业的传承
    1. 传承技术
    2. 传承管理方式
    3. 传承企业文化
