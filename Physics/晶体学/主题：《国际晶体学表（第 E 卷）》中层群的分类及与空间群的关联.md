# 《国际晶体学表（第 E 卷）》中层群的分类及与空间群的关联

## 一、层群的分类体系

根据《国际晶体学表（第 E 卷）》（以下简称《表 E》）前几页内容，层群（Layer Groups）作为**三维亚周期群**（Subperiodic Groups）的核心类型之一，其分类围绕**晶格特性、晶体系统、点群对称性**三大维度展开，形成明确的层级体系，具体如下：

### 1. 核心定义：层群的本质

层群是**具有二维平移对称性的三维对称群**，即仅在两个维度（通常定义为 a、b 方向）存在周期性平移，第三个维度（c 方向）无平移或仅含有限对称操作（如反射、旋转），对应晶体中 “层状结构” 的对称性描述（如表面吸附层、薄膜材料）。

### 2. 分类维度 1：二维布拉菲系统（Bravais System）

层群的晶格基础是**二维布拉菲格子**，共 4 类，决定了层群的晶格对称性，具体如下表：



| 二维布拉菲系统         | 晶格特征                         | 对应层群数量                           | 典型晶格参数限制                                    |
| --------------- | ---------------------------- | -------------------------------- | ------------------------------------------- |
| 斜方（Oblique）     | primitive（P）晶格，无直角           | 2（triclinic 类）+5（monoclinic 类）=7 | $a≠b$，$\gamma≠90°$（$\gamma$为 a、b 夹角，常规取非锐角） |
| 矩形（Rectangular） | 含 primitive（P）和 centred（C）晶格 | 11（orthorhombic 类）               | $a≠b$，$\alpha=\beta=\gamma=90°$             |
| 正方形（Square）     | primitive（P）晶格，四方对称          | 16（tetragonal 类）                 | $a=b$，$\alpha=\beta=\gamma=90°$             |
| 六方（Hexagonal）   | primitive（P）晶格，六方对称          | 8（trigonal 类）+8（hexagonal 类）=16  | $a=b$，$\gamma=120°$，$\alpha=\beta=90°$      |

*注：层群共 80 种，上述分类覆盖所有类型，且晶格中心性（P/C）仅在矩形系统中存在差异。*

### 3. 分类维度 2：三维晶体系统（Crystal System）

层群虽为 “亚周期群”，但其**点群对称性与三维空间群的点群直接对应**，因此沿用三维晶体系统的分类框架（剔除立方系统，因层群无立方对称），具体如下：



| 三维晶体系统           | 对应层群的点群类型                      | 层群数量 | 关键对称元素             |
| ---------------- | ------------------------------ | ---- | ------------------ |
| 三斜（Triclinic）    | 1、$\overline{1}$（对称中心）         | 2    | 无高次轴，仅恒等或对称中心      |
| 单斜（Monoclinic）   | 2、m、2/m（2 次轴 + 镜面）             | 5    | 1 个 2 次轴或 1 个镜面    |
| 正交（Orthorhombic） | 222、2mm、mmm（3 个正交 2 次轴 / 镜面）   | 30   | 3 个相互垂直的 2 次轴或镜面   |
| 四方（Tetragonal）   | 4、$\overline{4}$、4/m、422、4mm 等 | 16   | 1 个 4 次轴（或 4 次反演轴） |
| 三角（Trigonal）     | 3、$\overline{3}$、32、3m 等       | 8    | 1 个 3 次轴（或 3 次反演轴） |
| 六方（Hexagonal）    | 6、$\overline{6}$、6/m、622、6mm 等 | 8    | 1 个 6 次轴（或 6 次反演轴） |

*例：层群 pmm2（L23）属于正交晶系 / 矩形布拉菲系统，点群为 mm2，含 2 次轴和两个正交镜面。*

### 4. 分类维度 3：几何晶类（Geometric Crystal Class）

层群的几何晶类由其**点群（Point Group）** 决定，即忽略平移对称性后剩余的 “点对称操作集合”。《表 E》明确层群共 27 个几何晶类，与三维空间群的点群一一对应，例如：



* 层群的点群$2/m$对应空间群的单斜晶类$C_{2h}$；

* 层群的点群$4/mmm$对应空间群的四方晶类$D_{4h}$。

几何晶类进一步关联层群的 “对称性高低”，例如高对称晶类（如$6/mmm$）对应的层群（如 p6/mmm，L80）包含更多对称操作。

## 二、层群与空间群的关联：从 “包含” 到 “投影”

《表 E》的核心设计理念之一是**将层群作为空间群的 “子群” 或 “投影衍生群”**，二者通过 “维度缩减” 和 “对称性继承” 建立紧密联系，具体体现在以下 4 个层面：

### 1. 层群是空间群的 “子群”：平移对称性的约束

空间群是**三维全周期群**（在 a、b、c 三个维度均有周期性平移），而层群可视为空间群在 “某一维度平移被抑制” 后的子群，即：



* 若空间群的平移向量为$(t_a, t_b, t_c)$（三维全周期），当$t_c→0$（或 c 方向无平移）时，空间群退化为层群；

* 例：空间群 Pmm2（三维正交群，平移向量$(a, b, c)$）若忽略 c 方向平移，其对称操作集合（2 次轴、镜面）与层群 pmm2（L23）完全一致，即 pmm2 是 Pmm2 的 “层状子群”。

《表 E》在 “Preface” 中明确：“层群可视为空间群在二维平移子空间上的限制，其对称操作均来自空间群的操作子集”。

### 2. 空间群的 “截面对称性” 对应层群

当用一个平面（如垂直于 c 轴的平面）“截取” 具有空间群对称性的晶体时，平面内的原子排列对称性即为**层群**，这一过程称为 “空间群的截面投影”，具体规则：



* 投影方向：通常沿空间群的某一晶格轴（如 c 轴），投影后二维平面的对称性由空间群在该平面内的操作决定；

* 例：空间群 P4mm（四方晶系，三维全周期）沿 c 轴投影后，平面内的对称性为层群 p4mm（L55），二者的点群（4mm）和镜面 / 旋转轴分布完全一致；

* 《表 E》第 5 部分 “Scanning of Space Groups” 进一步量化了这一关联：对所有 230 个空间群，均给出其不同截面平面对应的层群（如 “截面平面垂直于 \[001] 时的层群为 pXXX”）。

### 3. 符号体系的直接对应： Hermann-Mauguin 符号

层群与空间群共享**Hermann-Mauguin（国际符号）** 规则，仅通过 “晶格中心性字母” 区分维度特性，具体对应关系：



| 符号组成     | 空间群                         | 层群                                      | 说明                                        |
| -------- | --------------------------- | --------------------------------------- | ----------------------------------------- |
| 晶格中心性    | 大写字母（P、I、F、C 等，三维晶格）        | 小写字母（p、c，仅二维晶格）                         | 层群仅含 p（原始）、c（面心）两类，因二维无体心 / 底心晶格          |
| 对称元素顺序   | 按三维方向（\[001]、\[100]、\[110]） | 按 “二维周期方向 + 垂直方向”（\[001]、\[100]、\[010]） | 层群的符号顺序突出二维周期性，如 pmm2 中 “mm” 对应 a、b 方向的镜面 |
| 高次轴 / 镜面 | 相同符号（4、6、m、$\overline{3}$等） | 相同符号                                    | 如空间群 P6mm 与层群 p6mm 的高次轴（6）、镜面（m）符号一致      |

*例：空间群 P2/m11（单斜，三维）与层群 p2/m11（L14，单斜），仅晶格字母（P→p）不同，对称元素（2/m）完全一致。*

### 4. 对称操作与 Wyckoff 位置的继承

层群的对称操作（如反射、旋转、滑移面）和 Wyckoff 位置（等效点系）均来自空间群的 “二维限制”：



* 对称操作：空间群中平行于层平面（a-b 面）的操作（如镜面 m//(001)、2 次轴 //\[001]）会完整保留在层群中，垂直于层平面的操作（如 c 方向滑移面）则可能转化为层群的 “内部对称”；

* Wyckoff 位置：层群的等效点坐标（如 (x,y,0)）是空间群等效点（如 (x,y,z)）在 z=0 平面的投影，其多重性、位点对称性（Site Symmetry）与空间群对应位置一致。

《表 E》在 “1.2.11 Positions” 中强调：层群的 Wyckoff 位置计算方法与空间群相同，仅需忽略第三维度的平移贡献。

## 三、总结：层群与空间群的 “从属 - 衍生” 关系



| 关联维度  | 具体表现                               | 核心意义                    |
| ----- | ---------------------------------- | ----------------------- |
| 分类体系  | 层群沿用空间群的晶体系统、点群分类，仅通过二维晶格修正        | 统一晶体对称性描述逻辑，便于跨维度关联     |
| 对称性来源 | 层群是空间群的子群（抑制一维平移）或截面投影             | 解释层状材料与块体材料的对称性继承       |
